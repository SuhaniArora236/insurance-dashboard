# 💖 Medical Insurance Analytics Dashboard

An interactive Python-based GUI dashboard exploring how demographic and lifestyle choices directly impact healthcare insurance premiums. Designed with a custom pastel pink aesthetic!

## 🚀 Features
* **Interactive Filtering:** Filter records instantly by lifestyle attributes (smoker status) and region.
* **Dynamic Analytics:** Visualizes pricing discrepancies between smokers and non-smokers via customized Plotly graphs.
* **Key Metrics:** Real-time calculation of data counts, average BMI, average age, and total average charges.

## 🛠️ Tech Stack
* **Python** (Core logic)
* **Streamlit** (Web app framework & UI)
* **Plotly Express** (Interactive charts)
* **Pandas** (Data manipulation)

## 📊 Dataset Credit
Data sourced from the [US Health Insurance Dataset on Kaggle](https://www.kaggle.com/datasets/teertha/ushealthinsurancedataset).
